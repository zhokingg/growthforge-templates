Chat completion in 3 lines

Drop-in replacement for any LLM API. Switch providers without changing code.

// Initialize Nexus client
import { Nexus } from '@nexus/sdk'

const nexus = new Nexus({ apiKey: process.env.NEXUS_KEY })

const response = await nexus.chat.complete({
  model: 'auto',  // Auto-routes to best model
  messages: [{ role: 'user', content: 'Explain quantum computing' }]
})

Built-in RAG with vector search

Semantic search over your documents with one API call. No vector DB setup required.

// Index documents
await nexus.documents.create({
  collection: 'docs',
  documents: [{ content: '...', metadata: { source: 'wiki' } }]
})

// Query with semantic search
const results = await nexus.search.query({
  collection: 'docs',
  query: 'How does our refund policy work?',
  limit: 5
})

Multi-step AI agents

Build agents that reason, plan, and execute tools autonomously.

const agent = nexus.agents.create({
  name: 'support-bot',
  instructions: 'Help customers resolve issues',
  tools: ['search_docs', 'create_ticket', 'send_email']
})

const result = await agent.run({
  input: 'My subscription was charged twice this month'
})

Streaming responses

Token-by-token streaming with built-in backpressure handling.

const stream = await nexus.chat.stream({
  model: 'gpt-4',
  messages: [{ role: 'user', content: 'Write a haiku' }]
})

for await (const chunk of stream) {
  process.stdout.write(chunk.content)
}
